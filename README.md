# metafactory-cortex-adapter-mattermost

The **cortex mattermost surface adapter**, delivered as an installable plugin bundle.

Extracted from `the-metafactory/cortex` `src/adapters/mattermost/` per [ADR-0024](https://github.com/the-metafactory/cortex/blob/main/docs/adr/0024-pluggable-surface-adapters.md). cortex declares this bundle as an `arc-manifest.yaml` dependency; `arc upgrade cortex` installs it and the plugin loader loads it as a first-party adapter.

Naming: [component repo-naming standard](https://github.com/the-metafactory/compass/blob/main/standards/component-repo-naming.md).
